# Apply log threshold and compute averaged likelihoods

Apply log threshold and compute averaged likelihoods

## Usage

``` r
get_final_likelis(object, log_thresh, mode = "train")
```

## Arguments

- object:

  TimeTeller list object

- log_thresh:

  log threshold value

- mode:

  either 'train' or 'test'

## Value

Updated object with thresholded and averaged likelihoods
