# Nature-style colour palette

Provides qualitative, sequential, or diverging colour palettes suitable
for publication figures.

## Usage

``` r
nature_palette(n = NULL, type = "qualitative")
```

## Arguments

- n:

  number of colours to generate (NULL returns the base palette)

- type:

  one of 'qualitative', 'sequential_blue', or 'diverging'

## Value

character vector of hex colour codes
